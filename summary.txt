The clean architecture is implemented by organizing the application code into multiple layers 
which can contain one or more projects each with a specific responsibility.

Domain Layer = This layer is the backbone of the clean architecture and all other projects defined in other layers should depend on this layer. 
it contains domain entities.  

Application Layer =  it defines interfaces that are implemented by the outer layers. 
This layer contains interfaces , DTOs, mappers, validators, etc.

Infrastructure Layer = This layer contains the implementation of the interfaces defined in the Application layer. 
It can have multiple projects using different third party libraries or frameworks.

Presentation(API) Layer = This layer is responsible for presenting and handling user interaction with the system.
It includes controllers, and other web component.
